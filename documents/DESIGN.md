## テーブル定義書
### Usersテーブル
| カラム名        | 意味                  | データ型 | PK   | FK   | NOT NULL | AUTO_INCREMENT | 制約 |
| :-------------- | :-------------------- | :------- | :--- | :--- | :------- | :---- | :------ |
| id              | ユーザーID            | INT   | ◯   |      |    ◯    | ◯ |         |
| name            | ユーザー名            | VARCHAR(25)   |      |      | ◯  |       |         |
| email           | メールアドレス        | VARCHAR(100)   |      |      | ◯ |    |   UNIQUE      |
| password | パスワード  | CHAR(60)   |      |      | ◯       |       |         |
